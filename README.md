# Tetorisu（テトリス）

このゲームは Javaで作られたシンプルなテトリスゲームです。  
キーボードで操作して遊べます 💗

## 🎮 ゲームの遊び方

- **← →** ：左右に移動  
- **↓（ダウン）** ：落下スピードUP  
- **↑（アップ）** ：回転  
- **SPACE（スペース）** ：ゲーム開始

---

## 💾 ダウンロード方法

### ① GitHub から ZIPをダウンロード
右上の **Code（コード）** → **Download ZIP**

### ② 解凍してフォルダを開く

### ③ `Tetorisu.java` をコンパイル
PowerShell（パワーシェル）を開いてフォルダに移動してね：

---


### Java ファイルをまとめてコンパイル：

```javac -encoding UTF-8 *.java```

### ゲームを起動：

```java Tetorisu```
